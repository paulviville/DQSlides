Dual Quaternions

  • Nombres duaux
  • \[e = a + \epsilon b\]
  • \[ \epsilon ^2 = 0 \text{ et } \epsilon \neq 0\]
  • Addition:
    \((a + \epsilon b) + (c + \epsilon d) = (a+c) + \epsilon (b+d)\)
  • Multiplication:
    \((a + \epsilon b) * (c + \epsilon d) = (ac) + \epsilon (ad + bc)\)
  • Conjugé:
    \((a + \epsilon b)^* = a - \epsilon b\)
  • Norme:
    \(|(a + \epsilon b)| = \sqrt(a^2 + b^2)\)
