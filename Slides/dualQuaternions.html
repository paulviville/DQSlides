Dual Quaternions

\(\zeta = Q_r + \epsilon Q_t\)

  • Translation puis Rototation:
    \(\zeta(R,T)= R + \epsilon\frac{1}{2} R T\)
  • Rotation puis Translation:
    \(\zeta(R,T)= R + \epsilon\frac{1}{2} T R\)
  • Unitaire:
    Représente une rotation et translation
  • Addition: \( \zeta_1 + \zeta_2 = (Qr_1 + Qr_2) + \epsilon(Qt_1 + Qt_2)\)
  • Multiplication: \(\zeta_1 \zeta_2 = (Qr_1 Qr_2) + \epsilon(Qr_1 Qt_2 + Qt_1 Qr_2)\)
  • Multiplication Scalaire: \(s\zeta = sQr + \epsilon sQt\)
  • Conjugé: \(\zeta^* = Qr^* + \epsilon Qt^*\)
  • Norme: \(||\zeta|| = \zeta \zeta^*\)
